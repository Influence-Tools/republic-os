# Florida SB 7028 (2021) — OGSR/Data Processing Software

OGSR/Data Processing Software; Amending a provision relating to an exemption from public records requirements for certain data processing software obtained by an agency under a licensing agreement that prohibits its disclosure and which software is a trade secret and agency-produced data processing software that is sensitive; removing the public record exemption for certain trade secret information; removing the scheduled repeal of the exemption, etc.

**Enacted** — Laws of Florida, Chapter 2021-105.
