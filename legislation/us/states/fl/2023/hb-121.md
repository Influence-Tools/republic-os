# Florida HB 121 (2023) — Florida Kidcare Program Eligibility

Florida Kidcare Program Eligibility; Increases income eligibility threshold for coverage under Medikids program component; increases income eligibility threshold for coverage under Florida Kidcare program; requires applicant seeking coverage under program to provide certain documentation if eligibility cannot be verified using reliable data sources; requires that premiums for certain enrollees under Florida Kidcare program be based on tiered system of uniform premiums.

**Enacted** — Laws of Florida, Chapter 2023-277.
