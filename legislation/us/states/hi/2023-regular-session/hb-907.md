# Hawaii HB 907 (2023 Regular Session) — RELATING TO TELEHEALTH.

Temporarily allows for the reimbursement of services provided through telehealth via an interactive telecommunications system and two-way, real-time audio-only communications in certain circumstances. Defines "interactive telecommunications system". Repeals 12/31/2025. (SD2)
