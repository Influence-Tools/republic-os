# Hawaii SB 483 (2023 Regular Session) — RELATING TO THE UNIFORM PROBATE CODE.

Updates articles I through IV of the Uniform Probate Code to adjust for inflation, provide additional clarity, resolve issues that have arisen in probate practice, and address societal changes in familial relations. (CD1)
