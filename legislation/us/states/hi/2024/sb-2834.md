# Hawaii SB 2834 (2024) — RELATING TO THE LANDLORD-TENANT CODE.

Amends the Residential Landlord-Tenant Code to provide for a process to dispose of the tenant's personal property and surrender the dwelling unit to a landlord when there is a death of a tenant. Takes effect 11/1/2024. (CD1)
