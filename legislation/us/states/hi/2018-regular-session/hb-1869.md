# Hawaii HB 1869 (2018 Regular Session) — RELATING TO THE LANDLORD TENANT CODE.

Provides a specific exemption to the landlord-tenant code for a seller who occupies the residential real property after the transfer of the seller's ownership rights. Takes effect 11/1/2018. (SD1)
