# Hawaii HB 257 (2019 Regular Session) — RELATING TO HOMELESSNESS.

Authorizes the use of private lands for the Ohana Zones Pilot Program. Provides additional time for the implementation of the Ohana Zones Pilot Program to June 30, 2023. Extends the Emergency Department Homelessness Assessment Pilot Program and the Medical Respite Pilot Program to June 30, 2020. Appropriates funds. (HB257 CD1)
