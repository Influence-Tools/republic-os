# Hawaii SB 25 (2019 Regular Session) — RELATING TO INSURANCE.

Amends various portions of the Hawaii Insurance Code under title 24, Hawaii Revised Statutes, to update and improve existing Insurance Code provisions. Establishes requirements for proposed name changes and use of a trade name or assumed name, effective 10/1/2019. Until 7/1/2024, allows the Insurance Division to create stopgap measures to implement the NAIC Health Benefit Plan Network Access and Adequacy Model Act. (CD1)
