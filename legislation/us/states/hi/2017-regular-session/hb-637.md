# Hawaii HB 637 (2017 Regular Session) — RELATING TO THE STATE BUILDING CODES.

Requires the State Building Code Council to adopt codes or standards within two years of official publication, otherwise automatic adoption into Hawaii State Building Code will occur until such adoption is effectuated. Deletes requirement for Council adoption of new model building codes within 18 months of official publication date. Authorizes Council to receive private funds for code adoption. Appropriates funds to Council. (HB637 CD1)
