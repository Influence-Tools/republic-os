# Oregon SB 1066 (2025R1) — Relating to safety requirements for residential facilities; and declaring an emergency.

</b> </i>] [<i>Makes permanent</i>] <b>Extends</b> an exception for certain residential training homes and adult foster homes from state building code requirements to have installed an automatic sprinkler system. Declares an emergency, effective on passage.
