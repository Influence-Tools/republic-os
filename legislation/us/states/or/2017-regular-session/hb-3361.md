# Oregon HB 3361 (2017 Regular Session) — Relating to open data standards.

Directs State Chief Information Officer to appoint Chief Data Officer to maintain web portal for publication of data of state agencies. Directs Chief Data Officer to establish open data standard and publish technical standards manual for state agencies publishing data on portal.
