# Oregon SB 116 (2021 Regular Session) — Relating to private security providers.

Provides that it is unlawful for private security provider or entity that employs private security providers to possess or use equipment, vehicles, uniforms or titles that imply affiliation with public or private safety agency.<b> Exempts certain campus security providers.</b>
