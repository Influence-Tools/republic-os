# Oregon SB 636 (2021 Regular Session) — Relating to mailing of notices.

Provides that when state agency is required by statute to mail <b>certain</b> notice to <i>person</i>]<b> individual</b>, and agency has on file post office box address for <i>person</i>]<b> individual</b>, agency must mail notice to post office box address<b> if address is individual's last known address on file or address to which individual has requested that notices be sent</b>.
