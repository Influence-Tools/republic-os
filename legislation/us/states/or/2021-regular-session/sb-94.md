# Oregon SB 94 (2021 Regular Session) — Relating to the State Landscape Contractors Board.

Changes design requirement for practical skills tests offered by State Landscape Contractors Board. Deletes requirement that board provide physical location for practical skills testing. Deletes requirement that board conduct practical skills test within eight-hour period on single day.
