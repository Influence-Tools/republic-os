# Virginia HB 2762 (2019) — Firefighting foam management; use of foam that contains PFAS chemicals.

Prohibits, beginning July 1, 2021, the use for training purposes or for testing, with some exceptions, of a class B firefighting foam that contains intentionally added PFAS chemicals, as defined in the bill.
