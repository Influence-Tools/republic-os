# Virginia HB 1725 (2019) — Public school building security enhancements; compliance with Uniform Statewide Building Code, etc.

Requires each school board, in consultation with the local building official and the state or local fire marshal, to develop a procurement plan to ensure that all security enhancements to public school buildings are in compliance with the Uniform Statewide Building Code and Statewide Fire Prevention Code.
