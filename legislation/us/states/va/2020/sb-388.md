# Virginia SB 388 (2020) — Virginia Residential Landlord and Tenant Act; return of security deposit.

Virginia Residential Landlord and Tenant Act; return of security deposit. Requires the landlord to return the tenant's security deposit, minus any deductions or charges, within 45 days of the termination of the tenancy or the date the tenant vacates the dwelling unit, whichever occurs last. Under current law, the 45-day period to return the security deposit begins on the date of the termination of the tenancy. This bill is identical to HB 594.
