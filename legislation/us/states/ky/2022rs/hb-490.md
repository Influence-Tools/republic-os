# Kentucky HB 490 (2022RS) — AN ACT relating to transportation, making an appropriation therefor, and declaring an emergency.

Amend KRS 434.415 to define the terms "airbag," "counterfeit supplemental restraint system component," "nonfunctional airbag," and "supplemental restraint system"; prohibit the use of counterfeit airbags or components; set forth penalities for the use of counterfeit airbags or components; set forth exception.
