# New York A 163 (2019-2020) — Requires a system to allow the posting of candidate and contest information contained on each ballot style on the board of elections website

Requires the state board of elections to create a system to allow a copy of the candidate and contest information contained on each ballot style submitted by each county board of elections to be accessible to the public via web.
