# Colorado HB 21-1056 (2021A) — Cost Thresholds For Public Project Bidding Requirements

The requirements of the "Construction Bidding for Public Projects Act" (CBPPA) generally apply to a public project if the cost of the project is reasonably expected to exceed $500,000 for any fiscal year; except that a public project supervised by the department of transportation (CDOT) is subject to the requirements of the CBPPA if the cost of the project is reasonably expected to exceed $150,000 for any fiscal year. The act:Increases the lower cost amount for CDOT projects to $250,000, which means that the requirements of the CBPPA, including the requirement that CDOT prepare a bid estimate when it proposes to undertake a project itself rather than awarding the project to a contractor through competitive bidding, will apply to a CDOT project only if the cost of the project is reasonably expected to exceed $250,000 for any fiscal year;
