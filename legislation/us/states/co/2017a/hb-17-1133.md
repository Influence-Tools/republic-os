# Colorado HB 17-1133 (2017A) — Repeal SOS Secretary of State Annual Report UCC Uniform Commericial Code Filing-office Rules

Statutory Revision Committee. The bill repeals the requirement that the secretary of state annually report to the governor and legislature regarding filing-office rules promulgated under the 'Uniform Commercial Code - Secured Transactions'.(Note: This summary applies to the reengrossed version of this bill as introduced in the second house.)
