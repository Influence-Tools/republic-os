# Illinois HB 5014 (93rd) — PROP TX-REVISED ASSESSMENTS

Amends the Property Tax Code. Provides that if a county assessor revises an assessment after the complete assessment list is published, then the county assessor must publish a subsequent list of all the revised assessments for that year.
