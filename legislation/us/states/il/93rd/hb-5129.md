# Illinois HB 5129 (93rd) — ALTERNATE FUELS-DATABASE

Amends the Alternate Fuels Act. Provides that the Secretary of State shall compile a database of the flexible fuel vehicles in the State by zip code area. Provides that the database shall be completed and made available to the public in both print and electronic format by January 1, 2005. Effective immediately.
