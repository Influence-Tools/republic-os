# Illinois HB 2657 (93rd) — BUDGET IMPLEMENTATION-FY2004

Creates the FY2004 Budget Implementation Act. Provides that the purpose of the Act is to make the changes in State programs that are necessary to implement the Governor's FY2004 budget recommendations.
