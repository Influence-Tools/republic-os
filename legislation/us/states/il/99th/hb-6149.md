# Illinois HB 6149 (99th) — VEH CD-VETERANS' HOMES PLATES

Amends the State Finance Act. Creates the Illinois Veterans' Homes Fund as a special fund in the State Treasury. Amends the Illinois Vehicle Code to allow for the issuance of decals by Illinois Veterans' Homes. Provides for the original fee, renewal fees, and fee distribution for Universal special license plates with decals issued by Illinois Veterans' Homes.
