# Illinois HB 2547 (99th) — MOBILE HOME-VIN IF KNOWN

Amends the Mobile Home Local Services Tax Enforcement Act. Provides that the annual tax judgment, sale, redemption, and forfeiture record must contain the vehicle identification number, but only if known (currently, there is no exception to this requirement).
