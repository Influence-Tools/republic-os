# Illinois HB 4022 (97th) — VEH CD-TRAUMA PROGRAMS-RAIL

Amends the Illinois Vehicle Code. Provides that a Section concerning accident counseling shall apply to Class I and Class II (instead of only Class I) rail carriers. Provides that each Class I and Class II rail carrier must review and update its accident counseling or trauma program filing on an annual basis. Provides that programs may be filed either by mail or electronically via an email address designated by the Illinois Commerce Commission. Effective immediately.
