# Illinois HB 1574 (97th) — SEC DEP RETURN-LIEN HOLDER

Amends the Security Deposit Return Act. In a provision that requires a lessor to return a security deposit or prepaid rent to the lessee upon transfer of residential property, removes an exception for property acquired by a lienholder.
