# Illinois HB 2239 (96th) — BUDGET IMPLEMENTATION-FY2009

Creates the FY2009 Budget Implementation Act. Provides that the purpose of the Act is to make the changes in State programs that are necessary to implement the FY2009 budget. Effective immediately.
