# Illinois SB 3169 (96th) — ID CARDS--NON-EXISTENT ADDRESS

Amends the Illinois Identification Card Act. Provides that information concerning a non-existent address used to obtain an identification card is false information for the purposes of a particular provision of the Act. Amends the Illinois Vehicle Code. Provides that information concerning a non-existent address used to obtain a driver's license or permit is false information for the purposes of a particular provision of the Code. Also makes technical changes.
