# Illinois SB 1960 (94th) — CRIM CD-BAN YO YO WATERBALLS

Amends the Criminal Code of 1961. Prohibits the sale of yo-yo waterballs in the State. Provides that a violation is a business offense punishable by a $1,001 fine for each violation. Provides that each sale of a yo-yo waterball is a separate offense.
