# Illinois HB 1483 (94th) — CD CORR-SUPERVISION

Amends the Unified Code of Corrections. Provides that supervision is not available to a defendant charged with violating any provision of the Illinois Vehicle Code or a similar provision of a local ordinance that governs the movement of vehicles if, within the 12 months preceding the date of the defendant's arrest, the defendant has been assigned court supervision for a violation that governs the movement of vehicles under the Illinois Vehicle Code or a similar provision of a local ordinance.
