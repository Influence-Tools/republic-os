# Illinois HB 655 (94th) — CODE ENFORCEMENT RESPONSE

Amends the Counties Code and the Illinois Municipal Code. Provides that, if a notice of an ordinance violation requires the respondent to answer within a certain amount of time, the county or municipality must reply to the answer within the same amount of time.
