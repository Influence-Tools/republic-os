# Illinois HB 1088 (94th) — ID CARD FRAUD-PENALTY

Amends the Illinois Identification Card Act. Provides that a person convicted of certain fraudulent ID card violations when there are aggravating circumstances is guilty of a Class 3 felony (now, Class 4 felony) and for a second or subsequent violation is guilty of a Class 2 felony (now, Class 3 felony).
