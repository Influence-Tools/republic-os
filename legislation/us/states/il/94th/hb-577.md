# Illinois HB 577 (94th) — VEH CD-FIREFIGHTER PLATE FUND

Amends the Illinois Vehicle Code provision regarding Illinois Fire Fighters' license plates. Provides that all moneys deposited into the Illinois Fire Fighters' Memorial Fund shall be used exclusively for the purposes listed in the Vehicle Code provision.
