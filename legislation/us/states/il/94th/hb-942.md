# Illinois HB 942 (94th) — GRAIN CODE-ELECTRONIC RECEIPTS

Amends the Grain Code. Authorizes the use of warehouse receipts in electronic form. Provides that electronic receipts are as valid and enforceable as paper receipts. Provides that references in the Code to written or printed receipts include electronic receipts. Effective immediately.
