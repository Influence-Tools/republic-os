# Illinois HB 3544 (94th) — TREASURER-ATM LOCATIONS

Amends the State Treasurer Act. Provides that the Treasurer may enter into written agreements with financial institutions for the provision of automatic teller machine services at any location under the control of a State agency (now, State office buildings, State parks, and State tourism centers). Prohibits any State agency from procuring these services without the approval of the Treasurer.
