# Illinois HB 870 (94th) — CD CIV PRO-QUICK-TAKE

Amends the Code of Civil Procedure. Grants quick-take authority to Lake County for the specified properties for road improvements and bridge replacement projects. Grants quick-take authority to DeWitt County for the specified properties for road improvements projects. Effective immediately.
