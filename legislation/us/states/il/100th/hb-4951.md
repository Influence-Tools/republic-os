# Illinois HB 4951 (100th) — SECURITY DEPOSIT-LEASE COSTS

Amends the Security Deposit Return Act. Provides that for a written lease that specifies costs, the costs specified shall be for damage beyond normal wear and tear and reasonable to restore the leased premises to the same condition as at the time the lease began. Effective immediately.
