# Illinois HB 2897 (100th) — CD CORR-HELPING PAWS-VETERANS

Amends the Unified Code of Corrections. Provides that the Helping Paws Service Dog Program shall include training service dogs for veterans with post-traumatic stress disorder (PTSD) or depression. Defines "veteran".
