# Illinois HB 4653 (98th) — CRIM CD-DOMESTIC BATTERY

Amends the Criminal Code of 2012. Provides that the enhanced penalties for domestic battery if a person has had prior convictions for certain offense apply to prior convictions under the law of another jurisdiction for any offense which is substantially similar.
