# Illinois SB 2733 (95th) — TIF-HISTORIC LANDMARKS

Amends the Illinois Municipal Code. Provides that the term "redevelopment project cost" does not include costs in a redevelopment project area in which a historic resource is to be demolished, removed, or substantially modified. Effective immediately.
