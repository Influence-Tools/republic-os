# Illinois HB 845 (95th) — CRIM CD-RECKLESS HOMICIDE

Amends the Criminal Code of 1961. Provides that in cases involving reckless homicide resulting in the death of a peace officer, the penalty is a Class 1 (rather than a Class 3) felony.
