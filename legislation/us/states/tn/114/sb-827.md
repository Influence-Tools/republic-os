# Tennessee SB 827 (114) — Schools, Private - As enacted, prohibits the state board of education from requiring a private school that provides a fully online, self-paced educational program to comply with certain class size, school calendar, vaccination, and assessment requirements in order to operate in this state as an approved Category III nonpublic school. - Amends TCA Title 49.
