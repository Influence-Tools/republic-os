# Tennessee SB 2066 (113) — Day Care - As enacted, requires each child care agency in this state to have installed in each room of the child care agency where care is provided to children a carbon monoxide alarm that is listed according to the International Building Code and International Fire Code for the purpose of carbon monoxide detection; provides that a violation is a Class C misdemeanor and that each day on which a violation continues constitutes a separate offense. - Amends TCA Title 68, Chapter 120 and Title 71, Chapter 3.
