# Tennessee HB 752 (113) — Children - As enacted, creates a criminal offense of a foster parent from a kinship placement knowingly allowing a child in the foster parent's care to visit with the child's parent if the foster parent has knowledge of a current court order prohibiting the parent from visiting with the child, with a first violation being a Class C misdemeanor punishable by a fine only and a second or subsequent violation being a Class B misdemeanor. - Amends TCA Title 37 and Title 39.
