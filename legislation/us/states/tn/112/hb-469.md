# Tennessee HB 469 (112) — Unclaimed Property - As enacted, changes the date used to determine when property held in a pension or retirement account is presumed abandoned; changes, from $50 to $25, the threshold value for property to be included in the report due to the treasurer under the Uniform Unclaimed Property Act; changes the exception to allow the treasurer to waive the claim requirement and deliver the property directly to a person. - Amends TCA Title 66, Chapter 29.
