# California AB 1125 (20192020) — Animal Control Officer Standards Act.

Existing law, the Code Enforcement Officer Standards Act, requires the Board of Directors of the California Association of Code Enforcement Officers to develop and maintain standards for the designation of Certified Code Enforcement Officers, and to designate minimum training, qualifications, and experience requirements for applicants to qualify for that designation.
