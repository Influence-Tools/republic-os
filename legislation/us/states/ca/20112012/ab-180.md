# California AB 180 (20112012) — Education: academic performance.

Existing law requires the Superintendent of Public Instruction, with approval of the State Board of Education, to develop an Academic Performance Index (API) , as part of the Public School Performance Accountability Program, to measure the performance of schools, especially the academic performance of pupils. The API consists of a variety of indicators including specified achievement test scores, attendance rates, and graduation rates. Existing law requires the Superintendent, with approval of the state board, to develop an alternative accountability system for specified types of schools, including, among others, community day schools and continuation schools. Existing law allows these schools to receive an API score, but prohibits them from being included in the API rankings of schools.
