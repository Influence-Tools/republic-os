# California SB 1015 (20112012) — Taxation: administration.

(1) Existing law authorizes the state to issue a withholding order for taxes to collect a state tax liability, including any penalties, accrued interest, and costs, in accordance with certain procedures. Existing law defines "state tax liability" to mean an amount for which the state has a state tax lien created pursuant to specified provisions.
