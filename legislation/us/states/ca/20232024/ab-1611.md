# California AB 1611 (20232024) — Fish and Game Code: violations.

Existing law generally makes any violation of the Fish and Game Code or any rule, regulation, or order made or adopted under the code a misdemeanor. Existing law makes a violation of specified regulations and provisions of the code an infraction or a misdemeanor.
