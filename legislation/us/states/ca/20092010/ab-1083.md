# California AB 1083 (20092010) — Health facilities: security plans.

Under existing law, the State Department of Public Health licenses and regulates hospitals, as defined. Violation of these provisions is a crime. Existing law requires hospitals to conduct a security and safety assessment and, using the assessment, develop a security plan with measures to protect personnel, patients, and visitors from aggressive or violent behavior. Existing law requires the plan to include specified security considerations.
