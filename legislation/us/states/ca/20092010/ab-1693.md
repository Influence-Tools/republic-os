# California AB 1693 (20092010) — Building standards: code adoption cycle.

(1) The California Building Standards Law provides for the promulgation of building standards by state agencies by requiring all state agencies that adopt or propose adoption of any building standard to submit the building standard to the California Building Standards Commission for approval or adoption. Existing law requires that the commission receive proposed building standards from state agencies for consideration in an annual code adoption cycle.
