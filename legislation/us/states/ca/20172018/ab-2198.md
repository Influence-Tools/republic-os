# California AB 2198 (20172018) — State government: FI$Cal: transparency.

The Financial Information System for California (FI$Cal) Act establishes the FI$Cal system, a single integrated financial management system for the state. Existing law requires that the system include a state transparency component that allows the public to have access to expenditure data using an Internet Web site. Existing law limits that public access to information regarding General Fund and federal fund expenditure data.
