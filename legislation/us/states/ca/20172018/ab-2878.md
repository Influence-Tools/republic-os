# California AB 2878 (20172018) — Local control and accountability plans: annual goals: state priorities: family engagement.

(1) Existing law requires the governing board of each school district and each county board of education to adopt a local control and accountability plan using a template adopted by the State Board of Education. Existing law requires the local control and accountability plan to include a description of the annual goals to be achieved for each of certain state priorities, which include the implementation of the academic content and performance standards adopted by the state board, as specified, and the specific actions that will be taken to achieve the annual goals.
