# California SB 184 (20172018) — Social security number truncation program.

Existing law requires the county recorder of each county to establish a social security number truncation program in order to create a public record version of each official record, in an electronic format, and requires the social security number contained in the record to be truncated, as specified. Existing law requires these provisions to apply to official records recorded on or after January 1, 1980, as specified.
