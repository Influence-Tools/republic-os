# Indiana SB 125 (2022) — Technical corrections.

Addresses technical errors in the Indiana Code, including spelling, tabulation, formatting, grammatical, and cross reference issues. Removes "empty" chapters which are no longer in use. (The introduced version of this bill was prepared by the code revision commission.)
