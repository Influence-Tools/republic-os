# Indiana SB 35 (2024) — Technical corrections.

Addresses technical issues in the Indiana Code, including those related to spelling, tabulation, formatting, grammar, and cross-references. Repeals an empty chapter. Relocates, without change, provisions enacted at an incorrect citation or at conflicting citations. Resolves technical conflicts from the 2023 legislative session. Changes references from the auditor of state to the state comptroller, and provides directions for publication of affected provisions. Makes conforming amendments. (The introduced version of this bill was prepared by the code revision commission.)
