# Indiana SB 209 (2020) — Search warrants.

Provides that a warrant authorizing a search, testing, or other analysis of an item is deemed executed when the item is seized. Provides that a warrant return is sufficient if the return contains a statement indicating that the item was seized by a law enforcement officer.
