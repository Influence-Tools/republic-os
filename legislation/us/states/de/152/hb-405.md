# Delaware HB 405 (152) — AN ACT TO AMEND TITLE 10 OF THE DELAWARE CODE RELATING TO THE JUVENILE OFFENDER CIVIL CITATION PROGRAM.

This Act removes the requirement that a juvenile can only be referred to the the Juvenile Offender Civil Citation Program a second or subsequent time if the juvenile’s prior referral was for a different offense than the second referral, allowing a juvenile to be referred for the same offense if the juvenile is otherwise qualified.
