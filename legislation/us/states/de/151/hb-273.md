# Delaware HB 273 (151) — AN ACT TO AMEND TITLE 16 OF THE DELAWARE CODE RELATING TO SYNTHETIC CANNABINOIDS.

This Act updates the synthetic cannabinoids listed on Schedule I of the Uniform Controlled Substances Act by using broader language so that new synthetic cannabinoids created after the enactment of this Act fall within this definition without needing to be specifically listed.
