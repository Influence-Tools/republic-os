# Delaware SB 42 (151) — AN ACT TO AMEND TITLE 14 OF THE DELAWARE CODE RELATING TO THE EDUCATOR EVALUATION SYSTEM.

This Act suspends the educator evaluation system during the 2020-2021 school year and replaces it with an observation and feedback cycle that provides educators with coaching and support related to hybrid and remote learning practices.
