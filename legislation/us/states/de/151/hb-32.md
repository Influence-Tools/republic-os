# Delaware HB 32 (151) — AN ACT TO AMEND TITLE 16 OF THE DELAWARE CODE RELATING TO TRASH CONTAINERS ON HIGHWAYS.

This Act provides all law-enforcement officers the authority to enforce the existing labeling requirements for dumpsters that are in a public road or parking lot.
