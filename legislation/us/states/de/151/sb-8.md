# Delaware SB 8 (151) — AN ACT TO AMEND TITLE 11 OF THE DELAWARE CODE RELATING TO DESTRUCTIVE WEAPONS.

This Act defines machine gun as it is defined under federal law. Defining machine gun makes clear the types of weapons prohibited under Delaware law, including an auto sear, or “glock switch”, which transforms a semiautomatic gun into a machine gun.
