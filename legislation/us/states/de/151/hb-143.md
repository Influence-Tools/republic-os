# Delaware HB 143 (151) — AN ACT TO AMEND TITLE 4 OF THE DELAWARE CODE RELATING TO ALCOHOLIC LIQUORS.

This Act removes taproom from the establishments for which the Commissioner shall refuse to grant a license for the sale of alcohol when there is an existing licensed establishment of similar type within a specified distance. The minimum distance between licensed establishments of similar type is made applicable only to a store or establishment for consumption off premises.
