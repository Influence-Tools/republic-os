# Delaware HB 229 (149) — AN ACT TO AMEND TITLE 14 OF THE DELAWARE CODE RELATED TO UNUSED SICK LEAVE UPON SEPARATION OF SERVICE.

This bill concerns sick leave payout for retired State Troopers who became reemployed in public education. Specifically, this bill adds language to the code that would treat retired State Troopers the same as other educational employees as it relates to sick leave payment upon separation from educational employment.
