# Delaware SB 248 (150) — AN ACT TO AMEND TITLE 29 OF THE DELAWARE CODE RELATING TO THE DISABILITY INSURANCE PROGRAM.

The purpose of the Bill is to allow for the modifications to the Disability Insurance Program for state employees that were included in the Governor’s Twelfth Modification to the COVID-19 State of Emergency to continue until December 30, 2020 in the event that the State of Emergency ends prior to that date.
